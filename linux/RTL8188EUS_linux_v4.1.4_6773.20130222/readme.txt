		3.1 rtl8188EUS_linux_v4.1.4_6773.20130222.tar.gz
			Naming rule: rtlCHIPS_linux_vM.N.P_ssss.yyyymmdd.tar.gz
			where:
				CHIPS: supported chips
				M: Major version
				N: miNor version
				P: Patch number
				s: SVN number
				y: package year
				m: package month
				d: package day
